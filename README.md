# Meta Health Marketing Website

A modern, AI-focused marketing website for Meta Health - a comprehensive healthcare technology platform.

## 🚀 What is Meta Health?

Meta Health is a complete healthcare technology platform that provides:

- **Practice Management** - Full practice management for medical practices
- **Hospital Management** - Enterprise-grade hospital systems
- **Telemedicine** - Built-in video consultations
- **Remote Patient Monitoring** - Track patient vitals remotely
- **Medical Aid Billing** - Automated claims for South African market
- **AI Automation** - AI-powered clinical notes, billing, scheduling, and insights

## 🎨 Design

- Clean, professional healthcare design
- Blue/teal color palette
- Mobile responsive
- Modern SaaS landing page layout

## 🛠️ Tech Stack

- **Framework**: React + Vite
- **Styling**: Tailwind CSS
- **TypeScript**: Type-safe components
- **Deployment**: Vercel-ready

## 📦 Installation

```bash
npm install
```

## 🏃‍♂️ Development

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## 🏗️ Build

```bash
npm run build
```

## 🚀 Deploy to Vercel

1. Push to GitHub
2. Import project in Vercel
3. Deploy with default settings (Vite detected automatically)

Or use Vercel CLI:

```bash
npm install -g vercel
vercel
```
