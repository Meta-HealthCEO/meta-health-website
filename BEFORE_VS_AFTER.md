## Animation Inventory

### Implemented Animations
1. **gradientFlow** - Background gradient movement (15s infinite)
2. **gradientRotate** - 4-point gradient rotation (20s infinite)
3. **borderRotate** - Animated border gradients (6s infinite)
4. **float** - Vertical floating motion (6s infinite)
5. **pulseGlow** - Box-shadow pulsing (3s infinite)
6. **fadeInUp** - Element reveal from bottom (0.6s)
7. **fadeInLeft** - Element reveal from left (0.6s)
8. **fadeInRight** - Element reveal from right (0.6s)
9. **scaleIn** - Element scale reveal (0.6s)
10. **drawLine** - SVG path drawing (2s)
11. **dotPulse** - Connection dot pulsing (2s infinite)
12. **spin** - Orbital rotation (20s infinite)

## Visual Effects Breakdown

### Glassmorphism
- **Background**: rgba(30, 41, 59, 0.4)
- **Blur**: backdrop-filter: blur(20px)
- **Border**: 1px solid rgba(148, 163, 184, 0.1)
- **Applied to**: All cards, navbar, overlays

### Glow Effects
Multiple variants:
```css
.glow-blue {
  box-shadow: 
    0 0 20px rgba(59, 130, 246, 0.3),
    0 0 40px rgba(59, 130, 246, 0.1);
}

Hover intensifies:
  box-shadow: 
    0 0 30px rgba(59, 130, 246, 0.5),
    0 0 60px rgba(59, 130, 246, 0.2);
```

### Gradient Meshes
```css
background: linear-gradient(
  -45deg, 
  #3B82F6, #8B5CF6, #06B6D4, #3B82F6
);
background-size: 400% 400%;
animation: gradientFlow 15s ease infinite;
```
